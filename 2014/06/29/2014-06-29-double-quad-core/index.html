Processor Core's: Double vs Quad

2014-06-29
College Relics

Contrasting Double Core to a Quad Core; Intel® Celeron® Processor 1037U and Intel Core i7-4770R Processor

An interesting current trend is the development of multi core technology first introduced with the Intel Core Duo processor on 7/27/2006 just over a year after the single core Pentium 4 came out. Since then, multi core processor technology has seen a deluge of core configurations ranging from two to dozens of cores. Last year I purchased a very cheap laptop for $225 with an Intel Celeron dual core processor. The processor was about a third of that cost. For an extra $263 I could have bought the four core Intel i7-4770R processor. CpuBenchmark.com ranks the i7 with a performance index of 10,298 versus the Celeron at 1779, 5 x’s greater, using a detailed number of benchmarks. The difference between these two processors can be examined side-by-side using some of these common benchmarks.

Traditionally, common benchmarks included clock speed and RAM. I will include these benchmarks as well as newer benchmark standards such as type of RAM, core configuration, memory channels, memory bandwidth, cache memory, bus type, graphics integration, and multi-threading capabilities. All benchmarks vary between the two processors with the exception of the instruction set which is 64 bit each requiring a 64 bit operating system; and the system bus which transfers data at a rate of 5 GB/second in each.

The clock speed of the Celeron is 1.8GHz to the i7 3.2GZh. Jobs get done faster with the faster processor which means more jobs getting done and less wait time for any individual job with a low queue rank. The clock speed is magnified by the fact that the i7 has 4 cores versus 2 for the Celeron. Moreover, the i7 can multithread x2 for each core equating to 8 simultaneous threads. The Celeron can only handle a single thread per core.

The Celeron has 4MB of RAM but only 2MB of cache memory, contrasting to the 8MB RAM and 6MB for the i7. Both come with two memory channels capable of processing 25.6GB per second, which is the rate at which data can be read to or written from memory. However, since the i7 has twice as much RAM and 3x’s more cache memory, the i7 can process much faster once applications use up 4GB of initial Celeron memory.

Lastly, both processors have graphics processing (GPU) built into the CPU chip. The Celeron CPU has Intel’s initial implementation of CPU graphics capability called HD Graphics. The i7 contains Intel’s HD Graphics 4600, which according to Intel “perform(s) much higher than the original version” [1].
