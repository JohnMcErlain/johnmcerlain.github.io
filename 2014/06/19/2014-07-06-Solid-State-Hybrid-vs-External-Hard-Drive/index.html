Solid State Hybrid vs. External Hard Drive

2014-06-19
College Relics

Two highly competitive storage devices based on their speed of data file access are the Solid State Drive (SSD) and the external hard disk drive (HDD). Both types can be attached internally to the Serial ATA (SATA). For portability, both can be attached externally via a USB connection and both have a similar appearance viewed from the outside. That is where the similarity ends. The hard drive is composed of magnetized media on a disk which rotates below a read/write head. Both the read/write head and the disk move: the disk by rotation and the read/write head on a pivoting arm. A lot of data can be stored on an external HDD, over a TB (1000 GB).

By contrast, the SSD uses flash RAM which functions in place of moving parts such as a revolving magnetic disk drive. Flash memory uses EEPROM (electrically erasable programmable read-only memory) There are two main types of flash memory, which are named after the NAND and NOR logic gates. NAND type flash memory may be written and read in blocks (or pages) which are generally much smaller than the entire device. NOR type flash allows a single machine word (byte) to be written—to an erased location—or read independently (http://en.wikipedia.org/wiki/Flash_memory, n.d.). These are non- volatile memories which persist after the power is removed (unlike RAM). Contrary to popular rumor, the effect of data degradation due to constant erasure is negligible (http://www.storagereview.com/ssd_vs_hdd, n.d.). The SSD uses an embedded processor which controls read/write operations and therefore the speed of data access. Fast SSDs can access data at rates of 6 GB/s.

The benefit to choosing a HDD would be twofold: Cost and capacity. The same sized unit of a SSD would cost four times that of a HDD yet contain only a quarter of the data as the HDD. However, the SSD has a minimum of four distinct advantages over the HDD. First is less than half the power consumption, important consideration when running off a battery. Also, there is less heat build-up. Next, because of no moving parts, there is no noise and vibration. Moreover, moving parts tend to break when bumped or dropped in transit, a consideration when storing data off-site. A third consideration is that SSDs are safe from magnetic radiation whereas the HDD is not. This may also be a factor in transit to an off-site destination. The most significant advantage of the SSD, however, is speed. File opening speeds are 30 % faster while copy/write speeds can be 5 times faster. System boots can be twice as fast.

The trade-off between these two types of drives is speed vs. capacity. However, hybrids are offered where one can determine the best specifications to suit their needs.
